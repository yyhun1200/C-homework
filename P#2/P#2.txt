문제정의:
1.10번의 턴 지정하고 한 턴이 끝날 경우, 턴 감소, 모두 소진 시 컴퓨터 승리 메시지 출력
2.사용자의 input을 받아 정답, min/max를 확인하고 프로그램 종료 또는 힌트출력 
3.각 조건에 따라 정답을 맞출 경우 지정한 메세지 출력,
4.한번에 맞출 경우 One-shot 지정 메세지 출력,
5.키 값이 0 또는 1000일 경우 나누고, You got the extreme number was xxx출력하기
6.키값의 배수인지 아닌지 판별하고, XXX is a multiple of the key 출력하기

구현 방법:
1.int 형 변수 life를 선언하여, 매턴이 끝난 뒤 1씩 감소시킨다. 
  루프문(while)을 활용하여 턴수만큼 게임을 반복하고, 
  루프가 끝나면 컴퓨터 승리 메세지를 출력한다.

2.scanf함수를 사용해서 사용자의 input을 받아 int형 변수 input에 저장한다. 
  그리고 if문을 활용하여 key값과 input값을 비교하여 힌트를 출력, 또는 정답인지를 확인한다.

3,4. input과 key값이 같을 경우 먼저 조건문으로 새로운 분기를 만든다.
  그 이후 조건문의 중첩을 통하여 각각을 또 다시 분기한다. 
  정답을 한번에 맞추는 경우는 life의 값이 10일 경우로, 0과 1000의 경우를 논리 연산자 ||를 통해 묶어 둘중 하나가 참이 될 경우,
  이외로 그냥 정답을 맞추는 경우로 정의하여 각각의 메세지를 출력한다. 
  정답을 맞추는 경우 프로그램을 종료하여야 하기 때문에 return 0; 선언한다.

5.정답인 경우가 false인 경우들 중에서 key 값과 값을 비교하여 큰 값과 작은 값을 따로 조건문을 작성하고 힌트 메세지를 출력한다.

6.multiple경우를 확인하기 위해 정답인 경우와, 크거나 작은 힌트를 제공한 뒤에 조건문을 작성한다. 
  배수인지 아닌지를 판단하기 위하여 나머지를 활용한다. 
  나머지가 0일 경우 배수일경우라고 볼수 있기 때문에, key % input == 0이라고 조건문을 작성하고, 
  참이라면 xxx is a multiple of the key라는 메세지를 출력한다.
